Detailed Description

template<typename Ta, int N, typename Tb, int Cb>
class stan::math::matrix_exp_action_vari< Ta, N, Tb, Cb >

This is a subclass of the vari class for matrix exponential action exp(At) * B where A is a double NxN matrix and B is a NxCb matrix.

The class stores the structure of each matrix, the double values of A and B, and pointers to the varis for A and B if A or B is a var. It also instantiates and stores pointers to varis for all elements of A * B.
