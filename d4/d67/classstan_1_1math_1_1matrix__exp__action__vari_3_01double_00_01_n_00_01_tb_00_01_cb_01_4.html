Detailed Description

template<int N, typename Tb, int Cb>
class stan::math::matrix_exp_action_vari< double, N, Tb, Cb >

This is a subclass of the vari class for matrix exponential action exp(At) * B where A is an N by N matrix of double, B is N by Cb, and t is data(time)

The class stores the structure of each matrix, the double values of A and B, and pointers to the varis for A and B if B is a var. It also instantiates and stores pointers to varis for all elements of exp(At) * B.

Template Parameters
NRows and cols for matrix A, also rows for B
TbScalar type for matrix B
CbColumns for matrix B
