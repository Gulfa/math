Detailed Description

template<typename Ta, int N, int Cb>
class stan::math::matrix_exp_action_vari< Ta, N, double, Cb >

This is a subclass of the vari class for matrix exponential action exp(At) * B where A is an N by N matrix of var, B is N by Cb matrix of double, and t is data(time)

The class stores the structure of each matrix, the double values of A and B, and pointers to the varis for A and B if B is a var. It also instantiates and stores pointers to varis for all elements of exp(At) * B.

Template Parameters
TaScalar type for matrix A
NRows and cols for matrix A, also rows for B
CbColumns for matrix B
